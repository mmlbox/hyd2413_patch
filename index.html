HYDLIDE3 FMPAC Patch 

こちらのMGSを元に、MSX1版ハイドライド3のBGMをFMPACに対応するIPSパッチです。

DOWNLOAD

TYPE IPS DESCRIPTION
FM+PSG hyd3_fp2.ips BGMがFM音源、効果音がPSGで演奏されます。
OPや街など効果音が必要ない場所では、BGMがFM音源+PSGで演奏されます。
どれをダウンロードしてよいか分からない場合はこちらをご利用ください。
FM+RHYTHM hyd3_r.ips BGMが全曲FM音源のみで演奏されます。効果音はPSGで再生されます。
BGMのリズム音は"チャカポコ"になります。YM2413のファンの方にオススメ。
FM+NOISE hyd3_f.ips BGMが全曲FM音源のみで演奏されます。効果音はPSGで再生されます。
BGMのリズム音はノイズ風音色になります。
FM+PSG (NO SE) hyd3_fp1.ips BGMが全曲FM音源+PSGで演奏されます。効果音は再生されません。

注意事項

  • パッチはMSX1版専用です。MSX2版には対応していません。
  • パッチは無保証です。自己責任でご利用ください。
